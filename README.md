# 中文本土化模组

一个中文本土化模组

## 使用方法

即使不太可能发生冲突，还是建议卸载所有汉化模组，模组管理器安装就是搜索下载安装，手动安装就是下载BepInExPack并提取BepInExPack“文件夹内”的3个内容放到游戏根目录，将本模组、GameTranslator和FontPatcher三个下载好的压缩包内的BepInEx文件夹拖入游戏根目录，很简单

如果你游戏根目录已经有BepInEx文件夹不知道做啥，覆盖呗...或者删掉BepInEx文件夹（无脑）

对于整合包作者等高级用户，你可以自行添加你想要的汉化规则，但注意区别在于，Normal的翻译要求相当精确的文本（无论字符是否在游戏内甚至在控制台内可见）

## 注意事项

当你安装上汉化模组后，你的游戏会被标记为Modded（模组）客户端，这是因为汉化模组使用了BepInEx模组加载器，我鼓励任何老玩家尝试使用原版进行游戏，即使那会导致无法查看中文字体以及看的全是英文（我个人不太愿意去直接改游戏文件来植入汉化，希望ZeekerssRBLX能给力点）

如果你有多人模组，正常情况下你遇到的都是Modded（模组）客户端/主机，大厅筛选的时候也不会显示筛选MOD大厅的选项，如果遇到加入不了其他人的房间并提示版本号不匹配，请检查你安装的模组
